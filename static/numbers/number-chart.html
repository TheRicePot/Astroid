Suggestions

You can use this chart to help you with your counting. Try counting backwards!

You can also try:

  • Make Odd Numbers one color and Even Numbers another
  • Color in by 3s (3, 6, 9, 12, 15, 18, 21, 24,...)
  • Color in by 4s (4, 8, 12, 16, 20, 24, 28, 32, ...)
  • Color in by 5s (5, 10, 15, 20, 25, ...)
  • Color in by 6s (6, 12, 18, 24, 30, 36, ...) then color the 3s in between
  • Start at 3 and color in each 5th one.
  • Make stripy patterns and see what numbers get colored
  • Make each row (rows go left-right) a different color
  • Make each column (columns go up-down) a different color
