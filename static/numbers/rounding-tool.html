Rounding Tool

Here you can Round Numbers using different Rounding Methods


Explanation

You can type in your own numbers or generate random numbers using the From, To and Count boxes.

The Decimals box allows you to change the number of decimal places you want to round to:

  • "1" rounds to tenths, "2" rounds to hundredths.
  • "-1" rounds to tens, "-2" rounds to hundreds.
